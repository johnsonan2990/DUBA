﻿ RRExplore

RoundRobinExplore

To use the jar file:

  • navigate to the folder with the jar (DUBA)
  • type in terminal: “java -jar RRExplore.jar” followed by whatever flags you want to use. You have to provide at least a filepath.

Available flags:

  • “-in”: use to specify filepath, absolute or relative from where the jar is placed.
  • “-slice”: use to specify slice size. Defaults to 10.
  • “-rounds”: use to specify number of rounds. Defaults to 20.
  • “-bound”: use to specify the stack bound to use for the overapproximation. defaults to 2.
  • “-delay”: use to specify a delay bound to explore up to. This does not do convergence tests. If this is not included, the program will run the version with interaction/convergence test.
  • “-automatic”: use if you don’t want the program to ask you to continue at each k increment.
